Belajar HTML

HTML (Hyper-Text Markup Language) adalah bahasa pemrograman web paling dasar. Belajarnya mudah dan cepat. Tidak ada perhitungan logika dan algoritma. Hanya menampilkan konten saja.

HTML dapat diedit dengan menggunakan berbagai aplikasi dasar seperti Windows Notepad, atau text editor seperti Visual Studio Code, Notepad++, dan Atom (IDE). Format filenya adalah .html dapat juga diedit secara online dengan bantuan online text editor/compiler seperti W3schools.com . File HTML dapat dibuka dengan bantuan web browser, seperti Google Chrome, Microsoft Edge, dan Mozilla Firefox.

HTML sangat mudah dikenali dengan syntax/aturan penulisannya yang menggunakan angle brackets yaitu < dan >. Contohnya adalah

<!DOCTYPE html>
<html lang="en">

</html>
